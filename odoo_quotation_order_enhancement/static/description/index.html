Quotation and Order Enhancement

This module shows discounted unit price and delivery date to order/Quotation report printout.

Configuration

Under Pricing in Sale > Configuration > Settings, check Discount.

In Quotations, add a discount.

Also in Other info, add a delivery date.

After print Quotation/Order, you can see Discount unit price and Delivery date.
